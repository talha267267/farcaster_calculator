
Farcaster Calculator Mini App - Vercel Deployment Guide

1. Upload all files (index.html, style.css, script.js) to a folder.
2. Install Vercel CLI: npm install -g vercel
3. Open terminal in project folder and run: vercel
4. Follow prompts (login, project name, framework: Other)
5. After deploy, Vercel will give a URL like: https://yourproject.vercel.app
6. Use this URL in Farcaster - Farcaster will detect <meta fc:frame> and display the mini app.

Optional: Replace <div class="ad-placeholder"> with your ad code (AdSense, affiliate banners, etc.)
